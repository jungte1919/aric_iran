시각화 설정

분석 기간 및 AISHub 사용자명을 입력하면 실시간 데이터가 적용됩니다 (선택사항).

분석 차트

탭을 눌러 차트를 전환합니다. 데모 데이터는 실제 통계 기반으로 생성됩니다.
* 데모 데이터는 실제 AIS 통계(WorldwideAIS.org, MarineTraffic 공개 보고서)를 기반으로 생성됩니다. AISHub 계정 입력 시 실시간 호르무즈 해협 탐지 선박 데이터로 교체됩니다.

사용 가능한 API 및 데이터 소스

각 API의 비용·조건 및 접근 방법을 확인하세요.

🛰 AISHub

무료 (조건부)
  • 실시간 선박 위치 (JSON/XML/CSV)
  • 지역 경계 박스 쿼리 가능
  • 조건: AIS 수신기 보유 또는 파트너 공유
  • 1분당 1회 요청 제한
  • 가입 신청 →

📡 aisstream.io

무료 티어 있음
  • 실시간 WebSocket AIS 스트림
  • 경계 박스 구독 방식
  • 무료: 일부 선박 유형·지역 제한
  • 선박 위치·속도·선종 수신
  • API 키 발급 →

⚡ EIA (에너지정보청)

완전 무료
  • 호르무즈 통과 석유 유량 (월별)
  • 백만 배럴/일 단위 통계
  • STEO 시리즈: PATC_HORMUZ
  • 최대 36개월 이력 제공
  • 무료 키 발급 →

🗺 MarineTraffic API

유료
  • 가장 포괄적인 AIS 이력 데이터
  • 지역 통계·국적·선종 집계 지원
  • CS1 플랜: 월 $50~부터
  • 최대 190일 이력 조회 가능
  • 플랜 확인 →

🔍 VesselFinder API

유료
  • 실시간·이력 선박 위치 데이터
  • 선박 세부 정보 (IMO, 국기, 선종)
  • 상업 플랜별 쿼리 수 상이
  • 문서 확인 →
