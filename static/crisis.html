실제 시세를 가져오지 못해 시각화용 샘플 데이터를 표시합니다. 실제 데이터를 보려면 pip install yfinance 후 서버를 재시작하세요.

yfinance 미설치

위기 진단 데이터를 보려면 가상환경에 yfinance를 설치한 뒤 서버를 재시작하세요.

.\.venv\Scripts\python.exe -m pip install yfinance

또는 install_yfinance.bat을 받아 프로젝트 폴더에 저장한 뒤 더블클릭해 설치할 수 있습니다.

Ctrl+클릭 복수 선택. 비우면 전체.

원자재 가격 (유가·귀금속)

노트북 유가변동과 동일: WTI, Brent Crude, Natural Gas, Copper, Gold, Silver.

가격 추세

12개월 전 대비 지수 (100=기준)

금융시장의 공포지수

노트북 금융시장의 공포와 동일: VIX(미국), MOVE(채권), SKEW, VVIX. 빨간 점선: 위험 기준.

외환 변동

노트북 외환변동과 동일: USD/KRW(달러/원), USD/CNY(달러/위안), USD/EUR(달러/유로), USD/GBP(달러/파운드).

주식시장 변동

노트북 주식시장 변동과 동일: S&P 500, TA-125(이스라엘), KOSPI, Nikkei 225 등 국가별 대표 지수.

지수 추세

12개월 전 대비 지수 (100=기준)

종가·지수값.
